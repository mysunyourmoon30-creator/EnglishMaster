# v0.3.0 Priority Score

## Scoring Formula

Weighted priority score:

```text
(User value x 3)
+ (Business value x 3)
+ (Fit with current system x 2)
- (Technical complexity x 2)
- (Security risk x 2)
- (Operational risk x 2)
- (Development effort x 2)
- Dependency risk
- Maintenance cost
- (Release risk x 2)
```

Higher score is better.

## Interpretation

The strict weighted score favors low-risk work that fits the current codebase. The final roadmap recommendation balances this score with strategic value:

- Content Authoring Improvement and Certificate System are the strongest low-risk product bets.
- External Email Provider is strategically important because it unlocks notifications and future account flows.
- Advanced Analytics is a good follow-up if scope remains small.
- AI Tutor should be discovery/preparation only unless risk controls are explicitly funded.
