# v0.4.0 Scope Lock

## Status

`Locked - Approved`

## Objective

Define candidate v0.4.0 scope without starting implementation.

## Candidate Scope

- Production hardening and operational automation.
- Certificate verification abuse protection and route-specific rate limiting.
- Analytics performance rollups if production measurements justify them.
- Dedicated public detail pages for learner-facing content.
- Expanded notification provider options and delivery observability.
- More complete learner progress capture across content workflows.

## Explicitly Out Of Scope Until Approved

- Payment/subscription flow.
- Marketplace.
- Mobile app.
- Multi-tenant SaaS conversion.
- AI tutor.
- Social/leaderboard features.

## Approval Gate

v0.4.0 implementation must not start until the release owner approves this scope lock or provides a revised scope.

Approved by the release owner on 2026-07-18. v0.4.0 implementation may proceed under the candidate scope defined above.
