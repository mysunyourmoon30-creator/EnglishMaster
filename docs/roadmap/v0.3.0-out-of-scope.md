# v0.3.0 Out Of Scope

## Won't Have

- AI Tutor full implementation.
- Mobile app.
- Payment/subscription flow.
- Marketplace.
- External search engine.
- Leaderboard/social features.
- Multi-tenant SaaS.
- Native mobile app.
- Full analytics warehouse.
- Broad collaboration suite.

## Allowed Preparation Only

- AI Tutor preparation may include discovery, safety analysis, cost modeling, grounding strategy, and evaluation planning.
- Mobile/PWA preparation may include responsive review for selected v0.3.0 flows.
- External search preparation may include requirements or indexing notes if content volume justifies it.

## Guardrails

- Do not add production code for out-of-scope themes in v0.3.0.
- Do not add migrations for out-of-scope themes.
- Do not expand v0.3.0 scope because a future theme is interesting.
- Promote any out-of-scope item only through explicit release-owner scope change.
