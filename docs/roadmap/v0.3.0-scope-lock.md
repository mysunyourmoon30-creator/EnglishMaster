## Explicitly Out Of Scope

- Multi-tenant SaaS.

## Implementation Rules

- Design prompts must run before implementation prompts.
- Review prompts must run before docs and stabilization prompts.
- Permission and privacy tests are required for user-facing or admin-facing endpoints.
- External provider secrets must come from environment variables or deployment secrets.
- Public endpoints must use public-safe DTOs only.
- Analytics queries must be bounded and privacy-safe.
- Durable data changes need migration review before release.
- Commit, tag, push, and deploy actions require explicit approval.
