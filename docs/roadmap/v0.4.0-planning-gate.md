Define the gate for opening v0.4.0 planning after the v0.3.0 release candidate path is documented.

- v0.3.0 RC tag exists.
- v0.3.0 UAT/staging plans exist.

- More complete learner progress capture.

## Guardrails

- Do not mix v0.4.0 feature work into v0.3.0 release branches.
- Do not use v0.4.0 planning to bypass unresolved v0.3.0 release gates.
- Keep security, permission, and data ownership checks as release blockers for new features.
