# Learning Recommendations Page

## Route

```text
/learn/recommendations
```

## Purpose

The recommendations page shows continue learning and rule-based suggestions grouped by content type.

## Recommendation Text

Reason text should remain simple and safe, for example:

- Continue where you left off.
- Matches your CEFR level.
- Not started yet.
- Recommended after low quiz score.
- Next lesson in your course.

## Known Limitations

- The UI is read-only.
- It does not expose detailed scoring math.
- It does not show AI-generated explanation text.
