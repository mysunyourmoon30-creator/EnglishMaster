# Learning Goals Page

## Routes

```text
/learn/goals
/learn/goals/create
/learn/goals/{id:guid}
```

## Goals List

The goals page shows:

- Active goals.
- Completed goals.
- Goal summary counts.
- Link to create a goal.
- Links to goal detail pages.

The page includes loading, empty, and error states.

## Create Goal

The create page lets a learner set a simple goal with:

- Goal type.
- Title.
- Description.
- Target value.
- Unit.

## Goal Detail

The detail page shows progress and lifecycle controls:

- Pause.
- Resume.
- Complete.
- Cancel.

## Known Limitations

- Goal progress is mostly manual.
- There are no charts yet.
- AI goal suggestions are intentionally excluded.
