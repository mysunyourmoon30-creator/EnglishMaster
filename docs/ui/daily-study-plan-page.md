# Daily Study Plan Page

## Routes

```text
/learn/study-plan
/learn/study-plan/history
```

## Today Page

The today page shows:

- Target minutes.
- Completed minutes.
- Completed item count.
- Plan status.
- Plan items ordered by priority.
- Generate today plan action.
- Complete item action.
- Skip item action.
- Complete plan action.

The page includes loading, empty, and error states.

## History Page

The history page lists recent daily study plans with:

- Plan date.
- Status.
- Completed item count.
- Completed minutes.

## Student Dashboard

The dashboard includes:

- Today's study plan section.
- Active goals summary.
- Study summary card.
- Generate today plan action.
- Start practice action.

## Known Limitations

- The plan is intentionally simple.
- Calendar integration is not included.
- AI plan generation is not included.
