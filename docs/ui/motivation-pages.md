# Motivation Pages

## Routes

```text
/learn/motivation
/learn/achievements
/learn/activity
```

## Motivation Page

Shows current streak, longest streak, achievement count, learning totals, recent achievements, and recent activity.

## Achievements Page

Shows earned, in-progress, and locked achievement progress. Learners can trigger evaluation.

## Activity Page

Shows recent learning activity with activity type, title, minutes, and date.

## Dashboard

The student dashboard includes streak and achievement summary cards.
