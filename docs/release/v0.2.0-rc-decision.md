## Known Limitations

Known limitations are documented in `docs/release/v0.2.0-known-limitations.md`.

Current release decision highlights:

- Several learner progress modules still need reviewed EF migrations before deployment.
- Learner activity capture is not wired into every workflow.
- Weekly reports are generated on demand, not by a scheduler.
- Public route shells rely on search where dedicated public detail pages are not yet implemented.
- Testing mode auto-authenticates non-security API routes, so manual auth/permission checks are required in UAT.
- Real staging/production secrets, monitoring, alerting, and external log sinks must be configured outside this repository.
