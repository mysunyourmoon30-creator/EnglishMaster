# v0.2.0 Known Limitations

## Product Scope

- No AI tutor yet.
- No mobile app.
- No payment or subscription flow.
- No marketplace.
- No external email provider yet.
- No external search engine.
- No advanced analytics or warehouse integration.
- No leaderboard or social features.

## Technical Limitations

- Several learner progress modules still need reviewed EF migrations before deployment.
- Learner activity capture is not wired into every workflow.
- Weekly reports are generated on demand, not by a scheduler.
- Public route shells rely on search where dedicated public detail pages are not yet implemented.
- Achievement definition admin editing UI is deferred.
- Import processing is synchronous and capped at 1 MB.
- Publishing runs synchronously and does not have a background worker or retry queue.
- Testing mode auto-authenticates non-security API routes, so manual auth/permission checks are required in UAT.

## Operations Limitations

- Real staging/production secrets must be supplied by the deployment environment.
- Production monitoring, alerting, and external log sinks must be configured outside this repository.
- Migration rollback must be planned carefully and backed by database backups.
